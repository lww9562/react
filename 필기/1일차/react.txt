npm (node package manager)

	npm init → package.json
	
	npm install	패키지명
	npm i		패키지명	→ 현재 프로젝트에만 설치(node_moduels)
		i -g		 	→ 전역 설치 : 모든 경로에서 실행 가능
		
	npm i -D	패키지명 	→ 개발 시에만 필요한 의존성 추가
							(Build 시 제거)
	
	npx 		명령어		→ 설치된 명령어를 실행
